# app

> My superb app made using [Electron](http://electron.atom.io/) and [Photon](http://photonkit.com/)


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
